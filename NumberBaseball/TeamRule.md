- step n - 닉네임에는 각자의 코드 작성
- step n 에는 PR 보낼 코드를 Merge!

  #### 🌱 Type 규칙

  - Feat ⭐️: 새로운 코드나 기능 추가

  - Refactor 📄 : 변수 이름을 바꾸거나, 코드 수정
  - Fix 🛠 : bug 수정(사용자 입장에서 느껴지는 것들)
  - Docs 📖 : 문서파일 추가 및 수정
